Bem vindo(a)!

O nosso sistema tem como objetivo auxiliar o público acadêmico em geral. Os usuários poderão salvar as imagens e usar para fins academicos. Para isso, disponibilizaremos imagens de alta qualidade de personalidades negras brasileiras que contribuiram de alguma forma para o país, mas foram “esquecidas”. Com isso, o sistema trás à tona discurssões importantes, como de cunho sociológico, filosófico e histórico.
